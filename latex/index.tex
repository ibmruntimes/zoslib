ZOSLIB is a z/\+OS C/\+C++ library. It is an extended implementation of the z/\+OS LE C Runtime Library. ~\newline


ZOSLIB implements the following\+:


\begin{DoxyItemize}
\item A subset of POSIX APIs that are not available in the LE C Runtime Library
\item EBCDIC \texorpdfstring{$<$}{<}-\/\texorpdfstring{$>$}{>} ASCII conversion C APIs
\item APIs for improved diagnostic reporting
\item and more!
